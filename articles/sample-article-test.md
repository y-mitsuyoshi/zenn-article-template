## はじめに

この記事は、Zenn記事執筆テンプレートのサンプルです。
このファイルを参考に、新しい記事を作成してください。

## このテンプレートでできること

- ローカル環境で快適に記事を執筆できます
- プルリクエスト作成時に自動で品質チェックが実行されます
- GitHub Copilotによる自動レビューを受けられます

## 記事の作成方法

新しい記事を作成するには、以下のコマンドを実行します。

```bash
npm run new:article
```

これにより、`articles/` ディレクトリに新しいMarkdownファイルが作成されます。

## ローカルプレビュー

執筆中の記事をブラウザでプレビューするには、以下のコマンドを実行します。

```bash
npm run dev
```

ブラウザで `http://localhost:8000` を開くと、記事のプレビューが表示されます。

## まとめ

このテンプレートを使って、効率的に高品質な技術記事を執筆しましょう!
